# Was ist Cloury?

Cloury ist ein modernes Minecraft-Netzwerk und Technologieprojekt.

Das Ziel ist die Entwicklung einer vollständigen Plattform mit:

- eigenem Framework
- eigenen Plugins
- eigener Infrastruktur
- Website
- Backend Services
- Cloud System
- zentralen Spielerprofilen

Cloury wird modular entwickelt und langfristig erweitert.

---

## Warum Cloury?

Der Name Cloury dient als zentrale Marke für das gesamte Projekt.

Er steht nicht nur für einen Minecraft-Server, sondern für das komplette Ökosystem aus:

- Netzwerk
- Cloud
- APIs
- Website
- Community-Systemen

---

## Ziele

- moderne Architektur
- hohe Performance
- Sicherheit
- Skalierbarkeit
