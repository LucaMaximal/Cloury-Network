# 🔄 Modular Sync

## Einführung

Das Modular Sync System gehört zu den wichtigsten technischen Komponenten von Cloury.

Es sorgt dafür, dass wichtige Daten eines Spielers auf dem gesamten Netzwerk aktuell bleiben.

Dadurch können verschiedene Server, Systeme und Plugins miteinander kommunizieren und Informationen austauschen.

---

> ⚠️ Hinweis
>
> Cloury befindet sich aktiv in Entwicklung.
>
> Die hier beschriebenen Funktionen stellen die aktuelle Planung dar und können sich jederzeit ändern oder erweitert werden.

---

# 🎯 Ziel des Systems

Ohne Synchronisation würden Server unabhängig voneinander arbeiten.

Dadurch könnten Probleme entstehen:

- unterschiedliche Coins
- falsche Statistiken
- veraltete Einstellungen
- nicht aktuelle Freundeslisten

Das Modular Sync System verhindert genau diese Probleme.

---

# 🌐 Netzwerkweite Daten

Viele Daten sollen auf allen Servern identisch sein.

Dazu gehören unter anderem:

- Coins
- Ränge
- Spracheinstellungen
- Profile
- Freundeslisten
- Achievements
- Level
- Statistiken

Ein Spieler soll unabhängig vom aktuellen Server dieselben Daten besitzen.

---

# ⚡ Schnelle Synchronisation

Langfristig soll die Synchronisation nahezu in Echtzeit erfolgen.

Das bedeutet:

Wenn ein Spieler beispielsweise Coins erhält, sollen diese sofort auf allen relevanten Servern verfügbar sein.

---

## Beispiel

Spieler A erhält 500 Coins.

Ohne Sync:

- Lobby zeigt 500 Coins
- Survival zeigt 0 Coins

Mit Modular Sync:

- Lobby zeigt 500 Coins
- Survival zeigt 500 Coins
- Website zeigt 500 Coins

Alle Systeme besitzen dieselben Informationen.

---

# 🔌 Modulares System

Das System wurde so geplant, dass jedes Plugin eigene Daten synchronisieren kann.

Beispiele:

- Rang

---

- Währungen

---

## Social

- Freunde
- Partys
- Clans

---

## Achievements

- Fortschritt
- Freigeschaltete Erfolge
- Belohnungen

---

# 🏗️ Warum Modular?

Jedes System arbeitet unabhängig.

Dadurch können neue Funktionen später leichter ergänzt werden.

Beispiele:

- neues Clan-System
- neues Cosmetics-System
- neue Spielmodi

Diese können das Sync-System nutzen, ohne komplett neu entwickelt werden zu müssen.

---

# 🌩️ Verbindung zur Cloury Cloud

Langfristig soll das Sync-System eng mit der geplanten Cloury Cloud verbunden werden.

Dadurch können Informationen zwischen:

- Servern
- Proxys
- Websites
- APIs
- Cloud-Diensten

ausgetauscht werden.
