# 🔌 Cloury Plugins

## Einführung

Cloury basiert nicht auf einem einzelnen riesigen Plugin.

Stattdessen wird das gesamte Netzwerk modular aufgebaut.

Das bedeutet, dass Funktionen auf mehrere Plugins aufgeteilt werden.

Dadurch wird die Entwicklung einfacher, Updates werden sicherer und neue Systeme können leichter ergänzt werden.

---

> ⚠️ Hinweis
>
> Cloury befindet sich aktiv in Entwicklung.
>
> Die hier beschriebenen Plugins stellen die aktuelle Planung dar.
>
> Funktionen, Namen, Strukturen und Systeme können sich jederzeit ändern, erweitert oder vollständig ersetzt werden.
>
> Ideen und Vorschläge können jederzeit über die Community-Bereiche eingereicht werden.

---

# 🏗️ Warum ein modulares System?

Viele Minecraft-Netzwerke verwenden wenige große Plugins.

Dadurch entstehen häufig Probleme:

- schwer wartbarer Code
- komplizierte Updates
- viele Abhängigkeiten
- schlechtere Erweiterbarkeit

Cloury verfolgt einen anderen Ansatz.

Jedes Plugin besitzt einen klar definierten Aufgabenbereich.

Dadurch können Systeme unabhängig voneinander entwickelt und erweitert werden.

---

# ⚙️ Das Core Plugin

## Cloury

Das Plugin **Cloury** bildet die Grundlage des gesamten Netzwerks.

Es enthält die wichtigsten Frameworks, APIs und Basissysteme.

Andere Plugins bauen auf diesem Plugin auf.

---

## Aufgaben des Core Plugins

Unter anderem:

- Framework Funktionen
- API Schnittstellen
- Datenverwaltung
- Modulverwaltung
- Netzwerkkommunikation
- Konfigurationen
- Spielerdaten
- Synchronisation

Ohne das Core Plugin können die meisten anderen Cloury Plugins nicht funktionieren.

---

# 📦 Main Plugins

Die Main Plugins bilden die wichtigsten sichtbaren Systeme des Netzwerks.

---

## Cloury-Chat

Verantwortlich für:

- Chat Nachrichten
- Formate
- Prefixe
- Nachrichten-Systeme
- Chat Funktionen

Geplante Funktionen:

- globale Chats
- private Nachrichten
- Chat Filter
- Chat Moderation

---

## Cloury-Player

Verantwortlich für:

- Spielerprofile
- Datenverwaltung
- Statistiken
- Einstellungen
- Fortschrittssysteme

Dieses Plugin verwaltet die meisten Informationen eines Spielers.

---

## Cloury-Interface

Verantwortlich für:

- GUIs
- Menüs
- Inventar-Systeme
- Navigation
- Benutzeroberflächen

Ziel ist eine einheitliche Bedienung aller Cloury Systeme.

---

## Cloury-Security

Verantwortlich für:

- Sicherheitsfunktionen
- Schutzsysteme
- Erkennung verdächtiger Aktivitäten
- Netzwerk-Sicherheit

Dieses Plugin soll langfristig verschiedene Schutzmechanismen bereitstellen.

---

## Cloury-Anticheat

Verantwortlich für:

- Cheat-Erkennung
- Analyse verdächtigen Verhaltens
- Schutz fairer Spielbedingungen

Das System wird kontinuierlich erweitert und verbessert.

---

## Cloury-Network

Verantwortlich für:

- Netzwerkverwaltung
- Serverkommunikation
- Synchronisation
- Proxy-Systeme

Dieses Plugin verbindet verschiedene Teile des Netzwerks miteinander.

---

## Cloury-Lobby

Verantwortlich für:

- Lobby Funktionen
- Navigation
- Server Auswahl
- Spieler Einstieg

Die Lobby dient als zentraler Startpunkt für alle Spieler.

# 🔄 Warum mehrere Plugins?

Mehrere Plugins bieten viele Vorteile.

## Bessere Wartbarkeit

Probleme können einfacher gefunden und behoben werden.

---

## Schnellere Entwicklung

Teams können an verschiedenen Systemen gleichzeitig arbeiten.

---

## Höhere Stabilität

Ein Fehler in einem Modul beeinflusst nicht automatisch das gesamte Netzwerk.

---

## Bessere Erweiterbarkeit

Neue Systeme können leichter ergänzt werden.

---

# 🌩️ Die Zukunft

Langfristig soll Cloury nicht nur ein Minecraft-Netzwerk sein.

Geplant ist ein vollständiges Ökosystem aus:

- Minecraft Netzwerk
- Website
- APIs
- Cloud Infrastruktur
- Community Plattformen
- Entwickler-Schnittstellen

Die Plugin-Struktur bildet dafür die technische Grundlage.
