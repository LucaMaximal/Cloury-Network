# 👤 Profile System

## Einführung

Das Profile System ist eines der wichtigsten Systeme innerhalb von Cloury.

Jeder Spieler besitzt ein eigenes Profil, das ihn im gesamten Netzwerk begleitet.

Dabei werden wichtige Informationen zentral gespeichert und serverübergreifend synchronisiert.

Das Profil bildet die Grundlage vieler anderer Systeme.

Dazu gehören unter anderem:

- Ränge
- Coins
- Einstellungen
- Freunde
- Statistiken
- Erfolge
- Spielzeit
- Fortschrittssysteme

---

> ⚠️ Hinweis
>
> Cloury befindet sich aktiv in Entwicklung.
>
> Die hier beschriebenen Funktionen stellen die aktuelle Planung dar und können sich jederzeit ändern.

---

# 🎯 Ziel des Systems

Das Ziel des Profile Systems ist es, alle wichtigen Spielerinformationen an einem zentralen Ort zu verwalten.

Dadurch erhält jeder Spieler ein einheitliches Erlebnis auf dem gesamten Netzwerk.

Egal auf welchem Server sich ein Spieler befindet – seine Daten bleiben erhalten.

---

# 📋 Gespeicherte Informationen

Ein Profil kann unter anderem folgende Daten enthalten:

## Allgemeine Informationen

- UUID
- Spielername
- Erstes Betreten
- Letzter Login
- Letzter Server

---

## Community Informationen

- Rang
- Prefix
- Freundesliste
- Clan
- Party Status

---

## Fortschritt

- Level
- Erfahrungspunkte (XP)
- Erfolge
- Spielzeit
- Statistiken

---

## Wirtschaft

- Coins
- Netzwerk-Währung
- Belohnungen

---

## Einstellungen

- Sprache
- Chat Einstellungen
- Sichtbarkeit
- Benachrichtigungen
- Persönliche Präferenzen

---

## Moderation

- Verwarnungen
- Mutes
- Bans
- Moderationsverlauf

---

# 🌐 Netzwerkweite Synchronisation

Profile werden nicht nur auf einem einzelnen Server verwendet.

Das System wurde dafür entwickelt, auf dem gesamten Netzwerk zu funktionieren.

Beispiele:

- Coins überall verfügbar
- Gleiche Spracheinstellungen
- Gleiche Freundesliste
- Gleiche Statistiken
- Gleicher Rang

Dadurch entsteht ein einheitliches Spielererlebnis.

---

# 📈 Level System

Geplant ist ein globales Level-System.

Spieler können Erfahrungspunkte sammeln und dadurch Level aufsteigen.

---

## Mögliche XP Quellen

- Spielzeit
- Achievements
- Events
- Quests
- Community Aktivitäten

---

## Vorteile

Level sollen langfristig verschiedene Belohnungen freischalten.

Beispiele:

- Coins
- Titel
- Cosmetics
- besondere Profilelemente
- zukünftige Belohnungen

---

# 🏆 Achievements

Das Profil ist eng mit dem Achievement System verbunden.

Spieler können verschiedene Erfolge freischalten.

Diese werden dauerhaft im Profil gespeichert.

---

## Beispiele

- Erste Spielstunde
- 100 Freunde
- Event Teilnahme
- Besondere Herausforderungen

---

# 🌍 Website Integration

Jeder Spieler erhält zukünftig ein eigenes Profil auf der Website.

Dort können verschiedene Informationen eingesehen werden.

---

## Geplante Inhalte

- Rang
- Level
- Coins
- Statistiken
- Spielzeit
- Achievements
- Freunde
- Badges

---

# 🛡️ Team Funktionen

Teammitglieder erhalten zusätzliche Möglichkeiten.

Je nach Berechtigung können sie:

- Profile ansehen
- Moderationsdaten prüfen
- Statistiken einsehen
- Probleme analysieren

Mögliche Beispiele:
