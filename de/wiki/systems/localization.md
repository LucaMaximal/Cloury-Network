# 🌍 Localization System

## Einführung

Das Localization System sorgt dafür, dass Cloury in verschiedenen Sprachen genutzt werden kann.

Ziel ist es, Spielern aus unterschiedlichen Ländern ein möglichst verständliches und angenehmes Spielerlebnis zu bieten.

Dabei werden Texte, Nachrichten, Menüs und viele weitere Inhalte automatisch in der gewählten Sprache angezeigt.

---

> ⚠️ Hinweis
>
> Cloury befindet sich aktiv in Entwicklung.
>
> Die hier beschriebenen Funktionen stellen die aktuelle Planung dar und können sich jederzeit ändern oder erweitert werden.

---

# 🎯 Ziel des Systems

Nicht jeder Spieler spricht dieselbe Sprache.

Deshalb soll Cloury langfristig die Möglichkeit bieten, Inhalte in mehreren Sprachen anzuzeigen.

Dadurch können Spieler das Netzwerk einfacher nutzen und besser verstehen.

---

# 🌐 Aktuell geplante Sprachen

Zum Start sind folgende Sprachen geplant:

- 🇩🇪 Deutsch
- 🇬🇧 Englisch

Diese beiden Sprachen bilden die Grundlage des Systems.

---

# 📋 Was wird übersetzt?

Langfristig sollen möglichst viele Inhalte übersetzt werden.

Beispiele:

- Menüs
- Chat Nachrichten
- System Meldungen
- Einstellungen
- Profile
- Website Inhalte
- Achievements
- zukünftige Spielmodi

---

# 👤 Persönliche Sprache

Jeder Spieler soll seine bevorzugte Sprache selbst auswählen können.

Die Sprache wird im Profil gespeichert.

Dadurch bleibt die Einstellung auf dem gesamten Netzwerk erhalten.

---

## Beispiele

Ein Spieler wählt Deutsch.

Alle unterstützten Systeme zeigen Inhalte auf Deutsch an.

Ein anderer Spieler wählt Englisch.

Dieselben Inhalte werden auf Englisch angezeigt.

---

# 🔄 Netzwerkweite Synchronisation

Die Spracheinstellungen sollen netzwerkweit synchronisiert werden.

Dadurch bleibt die gewählte Sprache erhalten, egal auf welchem Server sich ein Spieler befindet.

---

## Beispiele

- Lobby
- zukünftige Spielmodi
- Website
- Profile
- Menüs

verwenden dieselbe Spracheinstellung.

---

# 🌍 Weitere Sprachen

Grundsätzlich ist das System so aufgebaut, dass später weitere Sprachen hinzugefügt werden können.

Beispiele:

- 🇫🇷 Französisch
- 🇪🇸 Spanisch
- 🇮🇹 Italienisch
- 🇵🇱 Polnisch
- 🇹🇷 Türkisch

---

## Wichtiger Hinweis

Weitere Sprachen sind aktuell nicht fest geplant.

Neue Übersetzungen würden hauptsächlich dann entstehen, wenn:

- genügend Nachfrage besteht
- genügend Spieler die Sprache nutzen
- eine sinnvolle Wartung möglich ist

Die Entscheidung hängt daher stark von der zukünftigen Community ab.

---

# ⚙️ Technische Vorteile

Das Localization System bietet verschiedene Vorteile.

## Für Spieler

- bessere Verständlichkeit
- einfachere Bedienung
- angenehmeres Spielerlebnis

---

## Für Entwickler

- zentrale Übersetzungen
- einfache Erweiterbarkeit
- einheitliche Struktur

---

# 🌐 Website Integration

Langfristig soll auch die Website mehrere Sprachen unterstützen.

Dadurch können Spieler dieselbe Sprache sowohl im Spiel als auch auf der Website verwenden.

---

## Geplante Bereiche

- Wiki
- Profile
- Dokumentationen
- Einstellungen
- Community Seiten

---

# 🚀 Zukunft

Mit dem Wachstum von Cloury kann auch das Localization System erweitert werden.

Welche Sprachen zukünftig unterstützt werden, hängt vor allem von der Community und der tatsächlichen Nachfrage ab.

Unser Ziel ist es, möglichst vielen Spielern eine angenehme Nutzung zu ermöglichen, ohne unnötig komplexe Übersetzungen zu pflegen.

---
